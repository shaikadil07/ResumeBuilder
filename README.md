# 📄 Resume Builder Application

A Java-based Resume Builder application designed to help users create professional resumes quickly and efficiently. The system allows users to enter personal, educational, technical, and professional details and generate a structured resume format.

## 🚀 Features

### 👤 Personal Information

* Add personal details
* Contact information management
* Professional summary section

### 🎓 Education Details

* Add academic qualifications
* Store institution and graduation details
* Display educational background in a structured format

### 💻 Skills Management

* Technical skills
* Programming languages
* Tools and technologies

### 💼 Experience Section

* Internship details
* Work experience records
* Project experience

### 📄 Resume Generation

* Generate complete resume
* Organized and professional formatting
* Easy-to-read structure

## 🛠️ Technologies Used
